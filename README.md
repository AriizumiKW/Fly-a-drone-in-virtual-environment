# Fly a drone in virtual environment
 Contains stereo vision based self-driving (OpenCVSharp), and some other interesting features. Written within Unity3D/C#.
 
 Find OpenCVSharp here：https://github.com/shimat/opencvsharp

 
 简单地讲，这是一个Motion Planning + 三维重建， written within Unity3D/C#

 除了cv2.findfundamentalmatrix，基本上都是从头开始搭轮子。对初学者而言应该具有一定参考价值。
